## ? 为什么jsx return时需要加一个小括号  
在 JSX 中，返回时使用小括号是为了确保 JSX 代码能够正确地解析和渲染。这主要是为了防止在编写多行 JSX 代码时遇到自动插入分号的坑（Automatic Semicolon Insertion, ASI）。
