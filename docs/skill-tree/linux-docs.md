# Linux相关知识

## linux shell脚本语言有哪些
Bash（Bourne Again Shell）   
  它是 Bourne shell（/bin/sh）的增强版   

Korn shell（ksh）   
  结合了 Bourne shell 和 C shell 的一些特性   

C shell（csh）   
  它提供了一些与 C 语言相似的控制结构和语法   

Z shell（zsh）   
  它是对 Bourne shell 的扩展   
  
Fish shell   
  Fish shell 是一种用户友好的 shell 解释器，它提供了直观的语法和交互式提示，适合新手和有经验的用户使用。
